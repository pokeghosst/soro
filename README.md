> Super-focused monotasking task stack.

## Features

- Cool design!
- Only the topmost task in the stack is available - complete it to see what else is left.
- Tasks are stored locally with IndexedDB.

## Developing

```bash
bun i
bun dev
```

## Building for deployment

```bash
bun run build
```
